## VSCode开发环境配置

+ 下载xampp, 安装

+ VSCode Manage -> Settings -> Extensions -> PHP -> Edit in settings.json

  ```properties
  "php.validate.executablePath": "/opt/lampp/bin/php",
  "php.executablePath": "/opt/lampp/bin/php",
  ```

+ 安装PHP VSCode插件
  + PHP Debug
  + PHP Intelephense
  + Live Server
  + ......
